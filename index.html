S.S.S-2 Project Questions

Dahun/Mun ibere kan!

  1. Se iwadi nipa "AAYAN-OGBUFO" ki o si salaye lekunrere pelu apeere

  2. Tumo ewi yii si ede yoruba:

    1. If there is reward for the right
    2. And punishment for the wrong
    3. If there is commendation for the good
    4. And condemnation for evil
    5. If we sustain the honest
    6. And despise the corrupt
    7. Then justice will seem done
